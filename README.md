# swagger-sample
自动化生成API文档

## 说明  
基于`swagger-maven-plugin`、`swagger2markup-maven-plugin`、`asciidoctor-maven-plugin` 插件生成HTML、PDF格式API文档


1、  
``` xml
<plugin>
    <groupId>com.github.kongchen</groupId>
    <artifactId>swagger-maven-plugin</artifactId>
    <version>3.1.8</version>
</plugin>
```
